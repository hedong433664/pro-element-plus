## API

### Props

| 参数       | 说明             | 类型      | 默认值  |
| ---------- | ---------------- | --------- | ------- |
| `span`     | 栅格占据的列数   | `Number`  | `24`    |
| `use-grid` | 是否使用栅格布局 | `Boolean` | `true`  |
| `full`     | 是否填充满整行   | `Boolean` | `false` |
