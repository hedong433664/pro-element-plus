CheckboxGroup 多选框组 ​

配置化的多选框组。

基础用法 ​

使用 options 属性配置多选框组选项。

<template>
  <div class="demo-preview">
    <pro-checkbox-group v-model="value" :options="options" />
  </div>
</template>

<script lang="ts" setup>
import { ref } from "vue"
import type { ProCheckboxGroupOption } from "pro-element-plus"

const value = ref(["1"])
const options: ProCheckboxGroupOption[] = [
  { label: "选项 1", value: "1" },
  { label: "选项 2", value: "2" },
  { label: "选项 3", value: "3" }
]
</script>

<style lang="scss" scoped>
.demo-preview {
  display: flex;
  flex-direction: column;
  gap: 10px;
}
</style>

禁用 ​

使用 disabled 或 option.disabled 控制多选框组或某个选项禁用。

<template>
  <div class="demo-preview">
    <pro-checkbox-group v-model="value" :options="options" disabled />
    <pro-checkbox-group v-model="value2" :options="options2" />
  </div>
</template>

<script lang="ts" setup>
import { ref } from "vue"
import type { ProCheckboxGroupOption } from "pro-element-plus"

const value = ref(["1"])
const value2 = ref(["1"])

const options: ProCheckboxGroupOption[] = [
  { label: "选项 1", value: "1" },
  { label: "选项 2", value: "2" },
  { label: "选项 3", value: "3" }
]

const options2: ProCheckboxGroupOption[] = [
  { label: "选项 1", value: "1" },
  { label: "选项 2", value: "2", disabled: true },
  { label: "选项 3", value: "3" }
]
</script>

<template>

    <pro-checkbox-group

      use-check-all
      check-all-label="坤坤必须是全部"
    />
  </div>

<script lang="ts" setup>

import type { ProCheckboxGroupOption } from "pro-element-plus"

插槽 ​

使用插槽来自定义多选框项。

<template>
  <div class="demo-preview">
    <pro-checkbox-group v-model="value" :options="options">
      <template #default="{ option: { label, value } }">
        {{ value === "2" ? label + "插槽" : label }}
      </template>
    </pro-checkbox-group>
  </div>
</template>

<script lang="ts" setup>

CheckboxGroup Attributes ​

属性名类型默认值说明
model-value / v-modelstring[]|number[]-绑定值
valuestring[]|number[]-当前选中的值
optionsArray-多选框组选项
is-borderbooleanfalse是否显示边框
is-buttonbooleanfalse是否显示按钮样式
sizestringdefault尺寸，可选值为 default、large、small
label-keystringlabel选项 label 值，用于显示
value-keystringvalue选项 value 值，用于赋值
disabledbooleanfalse是否禁用
includeArray-包含的选项
excludeArray-排除的选项
use-check-allbooleanfalse是否使用全选
check-all-labelstring全选全选 label

事件名说明回调参数
